# System Prompt: "Liquid Glass" ArtCSV Landing Page

- Core Layout:
  - Create a 1600px max-width landing page hero section.
  - The background should be pure white with a subtle, layered gradient glow in the top-left (using blurred ellipses in soft sage blue #60B1FF and muted moss blue #319AFF).
  - The design must be fully responsive, transitioning from a single-column mobile view to a dual-column desktop layout.

- Typography:
  - Headlines & Brand: Use Fustat (Bold).
  - Body & UI: Use Inter (Normal/Medium).
  - SF Fonts: Incorporate SF fonts where appropriate to reinforce a refined, Apple-like interface feel.
  - Hero Headline: "制約のある環境でも、CSV編集は美しく速く。" (75px, 1.05 line-height, -2px tracking).

- Iconography:
  - SF Symbols: Use SF Symbols for icons where appropriate, ensuring a clean, modern, Apple-inspired interface language.

- The "Strong Liquid Glass" Navbar:
  - Position:
    - Sticky at top-[30px], centered, w-fit.
  - Visuals:
    - backdrop-blur-[64px], background rgba(255,255,255,0.16), rounded-[20px].
  - Fidelity Details:
    - Outer Stroke: 1px solid rgba(255,255,255,0.38).
    - Inner Highlight Shadow: inset 0px 1px 0px 0px rgba(255,255,255,0.65).
    - Soft Edge Shadow: 0px 8px 30px 0px rgba(15,23,42,0.08).
    - Backdrop Filter: Use an intense backdrop filter treatment to achieve a highly transparent, premium translucent glass look with soft light diffusion.
    - Surface Reflection: Add a subtle top-to-bottom reflective gradient overlay using rgba(255,255,255,0.22) to rgba(255,255,255,0.04).
    - Glass Tone: Keep the glass bright, clean, and colorless, avoiding gray muddiness or overly milky opacity.
  - Items:
    - Logo "ArtCSV" (Fustat), Nav links (ホーム, 特徴, 機能, FAQ), and a glassy "無料で試す" button with an arrow icon.

- Page Structure Order:
  - The page must follow a strict vertical document flow matching the nav links exactly.
  - Use this exact top-to-bottom order:
    1. ホーム
    2. 特徴
    3. 機能
    4. FAQ
  - Do not reorder, merge, or swap sections for aesthetics, responsiveness, or visual balance.

- Main Hero Content:
  - Layout:
    - Center the hero section both horizontally and vertically within the viewport.
  - Subheadline:
    - "厳しいセキュリティ環境を前提に設計した、HTML1ファイルのCSVツール。余計なインストールなしで、作成・編集・整形をシンプルに進められます。" (18px, Inter, -1px tracking).
  - Primary CTA:
    - "無料で試す" button.
  - Color:
    - #007AFF with backdrop-blur-[2px].
  - Details:
    - rounded-[16px], white text, inner highlight shadow inset 0px 4px 4px 0px rgba(255,255,255,0.35), and a white circular arrow icon.
  - Animation:
    - Scale 1.02 on hover with a smooth transition.

- Feature Grid:
  - Layout:
    - Display 4 feature blocks in a clean 2x2 grid.
  - Style:
    - Each feature block should match the liquid glass aesthetic.
  - Visuals:
    - Use glassmorphism styling with backdrop blur, soft transparency, rounded corners, subtle border, and inner highlight shadow.
  - Glass Semantic:
    - Define clear semantic roles for each glass surface so the navbar, feature cards, and CTA feel visually consistent and purpose-driven.
  - Glass Effects:
    - Apply premium glass effects such as layered transparency, soft highlights, subtle reflections, and depth to create a polished high-end UI.
  - Content:
    - Each block should present one key feature with a title and short supporting text.
  - Balance:
    - Maintain visual balance with the left hero content.
  - Responsiveness:
    - Ensure the section remains fully responsive.

- Key Technical Specs for the Developer:
  - Container:
    - Use a relative wrapper for the background glow and a z-10 main container for the content.
  - Smoothing:
    - Apply -webkit-font-smoothing: antialiased for the sharpest typography.
