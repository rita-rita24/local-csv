# LocalCSV Pro 機能設計書

## 1. データ管理機能

### 1.1 管理対象

本ツールは表形式データを2次元配列として保持する。各要素は文字列として扱う。

| 項目 | 内容 |
| --- | --- |
| データ本体 | 行配列の配列 |
| セル値 | すべて文字列で保持 |
| ヘッダー有無 | `hasHeader` で管理 |
| 区切り文字 | `,` `\t` `;` ` ` のいずれか |
| 文字コード | UTF-8、Shift-JIS、EUC-JP、ISO-2022-JP、UTF-16LE、UTF-16BE |
| BOM付与 | 真偽値で管理 |
| 改行コード | `crlf` または `lf` |
| クオート方式 | `auto` `all` `none` |
| 列幅 | 列インデックス単位で保持 |

### 1.2 正規化仕様

| 項目 | 挙動 |
| --- | --- |
| 行データ | 配列以外は無視する |
| セル値 | `null` `undefined` は空文字へ変換する |
| 列数不足 | 最大列数に合わせて空文字で補完する |
| 列幅 | 数値でない値は破棄し、最小幅 `60px` 未満は切り上げる |

### 1.3 ヘッダー行の扱い

| 状態 | 挙動 |
| --- | --- |
| ヘッダーOFF | 全行を本文データとして扱う |
| ヘッダーON | 1行目を列名とみなし、検索・並び替え・フィルタ・統計の本文対象から除外する |

### 1.4 保存対象状態

自動保存および復元の対象は以下とする。

| 項目 | 保存有無 |
| --- | --- |
| データ本体 | 保存する |
| ヘッダー有無 | 保存する |
| 区切り文字 | 保存する |
| 列幅 | 保存する |
| 文字コード | 保存する |
| BOM付与 | 保存する |
| 改行コード | 保存する |
| クオート方式 | 保存する |

## 2. 取込機能

### 2.1 クリップボード取込

#### 2.1.1 起点

| 起点 | 内容 |
| --- | --- |
| ペーストエリアクリック | クリップボードから読込 |
| ツールバー「貼り付け」 | クリップボードから読込 |
| データ未読込状態での `Ctrl/Cmd + V` | クリップボードから読込 |

#### 2.1.2 入力条件

| 項目 | 内容 |
| --- | --- |
| 入力形式 | プレーンテキスト |
| 想定区切り | カンマまたはタブ中心 |

#### 2.1.3 処理

1. クリップボード文字列を取得する。
2. 文字列が空白のみの場合はエラー終了する。
3. 先頭数行をサンプルとして区切り文字を推定する。
4. 推定区切り文字でCSV/TSV解析を行う。
5. 空行を除外し、列数を補完する。
6. 読込成功時は既存データを置き換える。
7. ヘッダー有無は `OFF` に初期化する。
8. 列幅は初期化する。
9. Undo/Redo履歴をクリアする。

#### 2.1.4 出力

| 項目 | 内容 |
| --- | --- |
| テーブル表示 | 読込内容で更新 |
| 区切り文字 | 自動判定値を反映 |
| 形式表示 | 区切り文字に応じて更新 |
| トースト | 読込行数を表示 |

#### 2.1.5 制約

| 項目 | 内容 |
| --- | --- |
| 文字コード判定 | クリップボード取込では行わない |
| 区切り自動判定 | セミコロン・スペース区切りは自動判定対象外寄り |
| 権限失敗 | クリップボードアクセス不可エラー表示 |

### 2.2 ファイル取込

#### 2.2.1 起点

| 起点 | 内容 |
| --- | --- |
| ツールバー「開く」 | ファイル選択を起動 |
| `Ctrl/Cmd + O` | ファイル選択を起動 |

#### 2.2.2 対応形式

| 形式 | 拡張子 |
| --- | --- |
| CSV | `.csv` |
| TSV | `.tsv` |
| テキスト | `.txt` |

#### 2.2.3 文字コード判定

ファイル読込時は以下の順序で文字コードを推定する。

1. BOM判定
2. 外部ライブラリによる文字コード推定
3. UTF-16BE/LEのヒューリスティック判定
4. UTF-8フォールバック

#### 2.2.4 インポートウィザード処理

| ステップ | 処理 |
| --- | --- |
| 判定 | 文字コード、区切り文字、行数/列数、ファイルサイズを表示 |
| 確認 | 指定条件でプレビュー表示 |
| 実行 | 条件確定後に本読込 |

#### 2.2.5 インポート条件

利用者は以下を変更できる。

| 項目 | 変更可否 |
| --- | --- |
| 区切り文字 | 変更可能 |
| 文字コード | 変更可能 |
| 1行目をヘッダーとして扱う | 変更可能 |
| プレビュー行数 | 10 / 20 / 50 行で変更可能 |

#### 2.2.6 読込確定時の処理

1. 指定文字コードでファイル内容をデコードする。
2. 指定区切り文字で全量解析する。
3. 解析結果が空ならエラー終了する。
4. `state.data` を更新する。
5. `state.hasHeader` をウィザード指定値へ更新する。
6. `state.encoding` を指定文字コードへ更新する。
7. 非対応文字コード出力しかできない場合はUTF-8へ切り替える。
8. ファイルハンドルがあれば保存先として保持する。
9. ファイル名バッジを更新する。
10. Undo/Redo履歴をクリアする。

#### 2.2.7 制約

| 項目 | 内容 |
| --- | --- |
| ファイル未選択 | 中断する |
| 空ファイル | エラー表示する |
| 解析不可 | エラー表示する |
| File System Access API 非対応 | input要素フォールバックで対応する |

### 2.3 CSV解析

#### 2.3.1 解析仕様

| 項目 | 内容 |
| --- | --- |
| クオート | `"` を囲み文字とする |
| エスケープ | `""` を `"` として復元する |
| 改行 | `\n` および `\r\n` を行区切りとして扱う |
| 空行 | 全セル空白の行は除外する |
| 列数不一致 | 最大列数に合わせて空セル補完する |

#### 2.3.2 非同期化条件

| 条件 | 挙動 |
| --- | --- |
| テキスト長が大きい | Web Workerで解析 |
| Worker利用不可 | メインスレッドで解析 |

## 3. 編集機能

### 3.1 セル直接編集

#### 3.1.1 起点

| 起点 | 内容 |
| --- | --- |
| セルクリック | セルへフォーカス |
| セル編集後のフォーカス離脱 | 編集内容確定 |
| `Enter` 移動 | 現セル確定後に下セルへ移動 |

#### 3.1.2 処理

1. セルは `contenteditable` で直接編集する。
2. フォーカス離脱時、変更前後を比較する。
3. 差分がある場合のみ履歴へ記録する。
4. セル値を更新する。
5. 統計を更新する。
6. 自動保存を予約する。

#### 3.1.3 制約

| 項目 | 内容 |
| --- | --- |
| IME変換中 | セル移動キーを抑制する |
| ヘッダー行 | ヘッダーON時は本文編集対象外 |

### 3.2 セル移動

| 操作 | 挙動 |
| --- | --- |
| `Tab` | 次セルへ移動 |
| `Shift + Tab` | 前セルへ移動 |
| `Enter` | 下セルへ移動 |
| `Alt + ↑ ↓ ← →` | 対応方向の隣接セルへ移動 |

仮想スクロール中に移動先が未描画の場合は、対象行を描画範囲内へスクロールしてからフォーカスする。

### 3.3 複数セル選択

#### 3.3.1 選択方法

| 方法 | 内容 |
| --- | --- |
| ドラッグ | 矩形範囲選択 |
| `Shift + クリック` | アンカーからの矩形選択 |
| `Ctrl/Cmd + A` | セルフォーカス中のみ全本文セル選択 |

#### 3.3.2 挙動

| 項目 | 内容 |
| --- | --- |
| 選択単位 | 常に矩形 |
| 単一セルクリック | 選択状態を残さず通常フォーカスへ戻す |
| 選択解除 | `Esc` |

### 3.4 選択セルコピー

#### 3.4.1 起点

| 起点 | 内容 |
| --- | --- |
| `Ctrl/Cmd + C` | 選択セルがある場合に有効 |

#### 3.4.2 処理

1. 選択セル群を行単位・列単位に並べ替える。
2. 選択された列だけをタブ区切り、行は改行区切りで整形する。
3. クリップボードへTSVとして書き込む。

#### 3.4.3 制約

| 項目 | 内容 |
| --- | --- |
| 疎な選択 | 行ごとに選択列のみを連結する |
| コピー失敗 | エラートースト表示 |

### 3.5 選択セル削除

#### 3.5.1 起点

| 起点 | 内容 |
| --- | --- |
| `Delete` / `Backspace` | 選択セルがある場合に有効 |

#### 3.5.2 処理

1. 対象セル群を取得する。
2. スナップショット履歴を保存する。
3. 対象セル値を空文字にする。
4. テーブル表示、統計、自動保存を更新する。

### 3.6 範囲貼り付け

#### 3.6.1 起点

| 起点 | 内容 |
| --- | --- |
| `Ctrl/Cmd + V` | 選択セルがある場合 |
| `Ctrl/Cmd + V` | 単一セルフォーカス中かつ複数セルデータの場合 |

#### 3.6.2 貼り付け規則

| 条件 | 挙動 |
| --- | --- |
| クリップボードが1セル | 選択セルすべてへ同値反映 |
| クリップボードが複数セル | 左上またはフォーカスセル起点へ展開 |
| テーブル範囲超過 | はみ出した分は無視 |

#### 3.6.3 処理

1. クリップボード文字列を行・列へ分解する。
2. スナップショット履歴を保存する。
3. 貼り付け先セルへ値を反映する。
4. 表示、統計、自動保存を更新する。

## 4. 履歴機能

### 4.1 Undo / Redo

#### 4.1.1 管理方式

| 項目 | 内容 |
| --- | --- |
| 履歴上限 | 20件 |
| 履歴種別 | スナップショット履歴、単一セル差分履歴 |

#### 4.1.2 スナップショット履歴対象

以下のような構造変化を伴う操作はスナップショット履歴で管理する。

- 行追加
- 列追加
- 行削除
- 列削除
- 行複製
- 列複製
- 並び替え
- 貼り付け
- 一括置換
- ヘッダー切替
- 行列移動
- 全クリア

#### 4.1.3 単一セル差分履歴対象

- セル直接編集

#### 4.1.4 操作

| 操作 | 挙動 |
| --- | --- |
| `Ctrl/Cmd + Z` | Undo |
| `Ctrl/Cmd + Y` | Redo |
| `Ctrl/Cmd + Shift + Z` | Redo |

## 5. 検索・置換機能

### 5.1 検索

#### 5.1.1 検索条件

| 条件 | 内容 |
| --- | --- |
| 検索文字列 | 必須 |
| 大文字小文字区別 | ON/OFF |
| 完全一致 | ON/OFF |
| 正規表現 | ON/OFF |
| 対象列 | 全列または特定列 |
| フィルタモード | 一致行のみ表示するか |

#### 5.1.2 一致判定

| モード | 判定方式 |
| --- | --- |
| 通常 | 部分一致 |
| 完全一致 | セル全体一致 |
| 正規表現 | `RegExp` による判定 |

#### 5.1.3 処理

1. 入力変更でデバウンス付き検索を実行する。
2. 検索条件に応じて一致セル一覧を生成する。
3. 一致件数を保持する。
4. 必要に応じて一致セルハイライト用マップを作る。
5. フィルタモード時は一致行セットを生成する。
6. 現在一致位置を先頭へ初期化する。

#### 5.1.4 表示規則

| 項目 | 内容 |
| --- | --- |
| 一致セル | ハイライト表示 |
| 現在一致 | 専用色で強調 |
| 結果欄 | `n / total件` 表示 |
| 検索中 | `検索中...` 表示 |
| 未一致 | `一致なし` 表示 |
| 正規表現不正 | `無効な正規表現` 表示 |

#### 5.1.5 大量データ対応

| 条件 | 挙動 |
| --- | --- |
| 行数 `12,000` 以上 | Search Workerを利用 |
| 行数が大きい | 行単位のキャッシュで前処理 |
| 一致数が極端に多い | 保存件数を上限で打ち切る |
| 行数 `20,000` 以上 | 全一致セルハイライトを制限する |

### 5.2 検索結果移動

| 操作 | 挙動 |
| --- | --- |
| 次へ | 次の一致へ移動 |
| 前へ | 前の一致へ移動 |
| `Enter` | 次の一致へ移動 |
| `Shift + Enter` | 前の一致へ移動 |

一致セルが現在表示外なら、そのセルが見える位置へ自動スクロールする。

### 5.3 検索履歴

| 項目 | 内容 |
| --- | --- |
| 保存条件 | 検索欄で `Enter` 実行時 |
| 上限 | 20件 |
| 表示方式 | datalist候補 |
| 永続化 | IndexedDB |

### 5.4 単一置換

#### 5.4.1 起点

| 起点 | 内容 |
| --- | --- |
| 置換ボタン | 現在一致位置を1件置換 |

#### 5.4.2 処理

1. 現在一致セルを特定する。
2. 検索条件に応じた置換用正規表現を生成する。
3. スナップショット履歴を保存する。
4. 対象セルだけを置換する。
5. 再検索を即時実行する。

### 5.5 一括置換

#### 5.5.1 対象範囲

| 条件 | 内容 |
| --- | --- |
| ヘッダーON | 2行目以降を対象 |
| 対象列指定あり | 指定列のみ対象 |
| 対象列指定なし | 全列対象 |

#### 5.5.2 処理

1. 置換用正規表現を生成する。
2. スナップショット履歴を保存する。
3. 行数 `5,000` 以上ならWorker利用を優先する。
4. 置換件数を集計する。
5. 再検索を即時実行する。

## 6. フィルタ機能

### 6.1 検索フィルタ

| 項目 | 内容 |
| --- | --- |
| 起点 | 詳細検索の「フィルタ（該当行のみ表示）」 |
| 条件 | 検索一致した本文行 |
| 結果 | 一致行のみ可視行へ残す |

フィルタOFF時は検索ハイライトだけを残し、行表示制御は解除する。

### 6.2 列フィルタ

#### 6.2.1 構成

1列ごとに以下の条件を保持できる。

| 条件種別 | 内容 |
| --- | --- |
| 値一覧条件 | チェックした値のみ許可 |
| 条件式 | 含む、等しい、以上など |
| 条件式ロジック | AND / OR |

#### 6.2.2 条件演算子

| 演算子 | 内容 |
| --- | --- |
| contains | 部分一致 |
| not_contains | 部分不一致 |
| equals | 完全一致 |
| not_equals | 完全不一致 |
| gte | 数値優先の以上比較 |
| lte | 数値優先の以下比較 |
| empty | 空文字 |
| not_empty | 非空文字 |

#### 6.2.3 適用仕様

| 結合単位 | 仕様 |
| --- | --- |
| 同一列内 | 値一覧条件と条件式の両方を評価 |
| 条件式同士 | AND または OR |
| 複数列間 | AND |

#### 6.2.4 値一覧生成

| 条件 | 挙動 |
| --- | --- |
| 通常行数 | メインスレッドで集計 |
| 行数 `8,000` 以上 | Filter Workerで集計 |
| ヘッダーON | 2行目以降で集計 |

### 6.3 検索フィルタと列フィルタの合成

可視行は以下の全条件を満たす行のみとする。

1. ヘッダーを除く本文行であること
2. 列フィルタを満たすこと
3. 検索フィルタON時は検索一致行であること

## 7. 行操作機能

### 7.1 行追加

| 機能 | 挿入位置 |
| --- | --- |
| 行追加 | 末尾 |
| 上に行を追加 | 対象行の直前 |
| 下に行を追加 | 対象行の直後 |

追加行の列数は現在列数に合わせ、全セル空文字で初期化する。

### 7.2 行コピー

対象行をタブ区切りで整形し、必要に応じてクオートしてクリップボードへ出力する。

### 7.3 行複製

対象行の内容を複製し、直下へ挿入する。

### 7.4 行削除

確認モーダルで承認後に対象行を削除する。最後の1行削除時はデータ全体を空状態へ戻す。

### 7.5 行移動

行メニューボタンをドラッグし、ドロップ位置の前後へ挿入する。移動後は表示と統計を再計算する。

## 8. 列操作機能

### 8.1 列追加

| 機能 | 挿入位置 |
| --- | --- |
| 列追加 | 末尾 |
| 左に列を追加 | 対象列の左 |
| 右に列を追加 | 対象列の右 |

追加列は全行へ空文字セルを挿入する。列幅は近傍列幅を引き継ぐ。

### 8.2 列コピー

対象列を改行区切りでクリップボードへ出力する。

### 8.3 列複製

対象列を右隣へ複製挿入する。列幅も複製する。

### 8.4 列削除

確認モーダルで承認後に対象列を削除する。最後の1列削除時はデータ全体を空状態へ戻す。

### 8.5 列並び替え

#### 8.5.1 起点

| 起点 | 内容 |
| --- | --- |
| 列メニュー「昇順に並び替え」 | 昇順ソート |
| 列メニュー「降順に並び替え」 | 降順ソート |

#### 8.5.2 比較仕様

| 条件 | 比較方法 |
| --- | --- |
| 両値が数値化可能 | 数値比較 |
| それ以外 | ロケール文字列比較 |

#### 8.5.3 対象範囲

| 状態 | 対象 |
| --- | --- |
| ヘッダーOFF | 全行 |
| ヘッダーON | 2行目以降のみ |

#### 8.5.4 非同期化条件

| 条件 | 挙動 |
| --- | --- |
| ソート対象行数 `5,000` 以上 | Worker利用を優先 |

### 8.6 列移動

列ヘッダーのドラッグハンドルを用いて列順序を変更する。列幅設定も移動後の列順に再配置する。

### 8.7 列幅変更

列リサイズハンドルのドラッグで列幅を変更する。

| 項目 | 内容 |
| --- | --- |
| 最小幅 | `60px` |
| 保存対象 | 自動保存対象 |
| 適用範囲 | ヘッダーと本文セルの両方 |

## 9. 保存・出力機能

### 9.1 上書き保存

#### 9.1.1 起点

| 起点 | 内容 |
| --- | --- |
| ツールバー「保存」 | 上書き保存または保存先選択 |
| `Ctrl/Cmd + S` | 同上 |

#### 9.1.2 処理

1. 現在の区切り文字、改行、クオート方式で出力文字列を生成する。
2. 保存先ファイルハンドルがあれば上書き保存する。
3. ハンドルが無ければ保存先選択を行う。
4. 失敗時はダウンロード保存へフォールバックすることがある。

### 9.2 簡易コピー

現在の区切り文字形式で全データを文字列化し、クリップボードへ出力する。

### 9.3 簡易ダウンロード

現在の区切り文字形式で全データをBlob化し、ダウンロードする。

### 9.4 出力文字列生成

#### 9.4.1 DSV生成規則

| 項目 | 内容 |
| --- | --- |
| `quoteStyle = all` | 全フィールドをクオート |
| `quoteStyle = none` | クオートしない |
| `quoteStyle = auto` | 区切り文字、`"`、改行を含む場合のみクオート |
| クオート内 `"` | `""` へエスケープ |
| 改行 | 設定値に従う |

#### 9.4.2 非同期化

行数が多い場合は一定件数ごとに処理を分割し、UI応答性を維持する。

### 9.5 エクスポート

#### 9.5.1 対応形式

| 形式 | 内容 |
| --- | --- |
| CSV | カンマ区切り |
| TSV | タブ区切り |
| TXT | スペース区切り |
| JSON | 配列またはオブジェクト配列 |
| NDJSON | 1行1JSON |
| HTML | `<table>` 形式 |
| Markdown | Markdownテーブル |

#### 9.5.2 出力対象

| 対象 | 内容 |
| --- | --- |
| 全件 | 全本文行 |
| フィルタ結果のみ | 現在可視な本文行 |
| 選択範囲のみ | 選択範囲の外接矩形 |

#### 9.5.3 ヘッダー出力

| 条件 | 挙動 |
| --- | --- |
| ヘッダーONかつヘッダー出力ON | 1行目をヘッダーとして含める |
| ヘッダーOFF | ヘッダー行は出力しない |

#### 9.5.4 空行・空列除外

| 項目 | 内容 |
| --- | --- |
| 空行除外 | すべて空文字の行を除外 |
| 空列除外 | 対象範囲内で全行空文字の列を除外 |

#### 9.5.5 JSON/NDJSON変換

| 条件 | 出力構造 |
| --- | --- |
| ヘッダーあり | 列名をキーとしたオブジェクト |
| ヘッダーなし | 行配列の配列または行配列単位 |

#### 9.5.6 プレビュー

| 項目 | 内容 |
| --- | --- |
| 最大行数 | 200行 |
| 更新契機 | 形式、対象、ヘッダー有無、空行/空列除外、改行、クオート、文字コード変更時 |

#### 9.5.7 非対応文字コード

外部ライブラリ無しでShift-JIS、EUC-JP、ISO-2022-JP出力が必要な場合は、UTF-8へ切り替えて案内する。

## 10. 設定機能

### 10.1 文字コード設定

保存、簡易ダウンロード、エクスポート時の既定文字コードとして扱う。

### 10.2 BOM設定

対応文字コードは以下のみとする。

- UTF-8
- UTF-16LE
- UTF-16BE

非対応文字コード選択時はチェックを無効化し、必要なら自動OFFとする。

### 10.3 区切り文字設定

| 項目 | 反映先 |
| --- | --- |
| 区切り文字 | 簡易コピー |
| 区切り文字 | 簡易保存 |
| 区切り文字 | データ形式表示 |
| 区切り文字 | ツールバーラベル |

### 10.4 改行コード設定

保存およびDSV系出力の改行コードに反映する。

### 10.5 クオート設定

保存およびDSV系出力のクオート規則に反映する。

### 10.6 ヘッダー切替

#### 10.6.1 起点

| 起点 | 内容 |
| --- | --- |
| 設定「1行目をヘッダーとして扱う」 | ON/OFF切替 |

#### 10.6.2 処理

1. スナップショット履歴を保存する。
2. `hasHeader` を反転する。
3. テーブル、検索対象列、統計を更新する。
4. 自動保存を予約する。

### 10.7 テーマ設定

| 項目 | 内容 |
| --- | --- |
| ライトモード | `colorScheme = light` |
| ダークモード | `colorScheme = dark` |

### 10.8 表示言語設定

日本語と英語を切り替える。静的テキストだけでなく、動的メッセージ、タイトル、プレースホルダーも翻訳対象とする。

### 10.9 アクセントカラー設定

プリセット6色から選択し、主要色、ホバー色、セルフォーカス背景色へ反映する。

## 11. 統計機能

### 11.1 集計項目

| 項目 | 内容 |
| --- | --- |
| 行数 | 本文行数 |
| 列数 | 列数 |
| サイズ | 現在設定で出力した場合の概算バイト数 |
| 形式 | 現在の区切り文字に応じた表示 |

### 11.2 サイズ計算

現在の区切り文字と文字コードで1行ずつエンコード長を積み上げる。

### 11.3 非同期化条件

| 条件 | 挙動 |
| --- | --- |
| 行数 `5,000` 超 | 分割計算 |
| 行数 `20,000` 以上 | デバウンス付きで実行 |

## 12. 永続化機能

### 12.1 保存先

| 項目 | 保存先 |
| --- | --- |
| 編集状態 | IndexedDB |
| 検索履歴 | IndexedDB |
| テーマ | IndexedDB |
| 言語 | IndexedDB |
| アクセントカラー | IndexedDB |

IndexedDB利用不可時は `localStorage` またはメモリへフォールバックする。

### 12.2 自動保存

| 条件 | デバウンス時間 |
| --- | --- |
| 通常 | 300ms |
| 大量行 | 1,200ms |
| 超大量行 | 2,600ms |

### 12.3 復元

初期化時に保存済み状態を読込み、データが存在する場合はそのまま編集再開できる状態へ戻す。

## 15. 例外・制約

| 項目 | 内容 |
| --- | --- |
| 右クリック | コンテキストメニューを無効化する |
| 正規表現不正 | 検索・置換は実行せずエラー表示する |
| 最後の1行/1列削除 | データ全体を空状態へ戻す |
| 非対応文字コード出力 | UTF-8へ切替える |
| 選択範囲エクスポート | 疎な選択をそのまま保持せず矩形化する |
| ヘッダー編集 | ヘッダーON中は直接編集できない |
