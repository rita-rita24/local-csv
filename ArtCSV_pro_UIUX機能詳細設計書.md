## 2. 画面状態

### 2.1 初期状態

- データテーブルは非表示。
- ペーストエリアが主表示となる。
- 統計情報は行数 `0`、列数 `0`、サイズ `0 KB` 相当の表示になる。
- ファイル名バッジは空。

### 2.2 データ読込後

- ペーストエリアは非表示になる。
- テーブル領域が表示される。
- 行数、列数、サイズ、形式が更新される。
- 列ヘッダー、行番号、各セルが操作可能になる。

### 2.3 フィルタ・検索状態

- 検索一致セルはハイライト表示される。
- 現在選択中の一致セルは別色で強調表示される。
- 列フィルタが有効な列はヘッダーのフィルタアイコンがアクティブ表示になる。
- 検索のフィルタモード有効時は一致行のみが表示対象になる。

### 2.4 モーダル状態

- 確認、設定、ヘルプ、エクスポート、インポートはオーバーレイ表示。
- オーバーレイ外クリックで閉じるモーダルがある。
- `Esc` キーで選択解除、検索解除、モーダル終了のいずれかが優先実行される。

## 3. 主要導線

### 3.1 クリップボードから新規データを取り込む

1. 利用者は空状態のペーストエリアをクリックするか、ツールバーの「貼り付け」を押す。
2. クリップボード文字列を読み取り、区切り文字を自動判定して解析する。
3. 解析後はテーブル表示に切り替わる。
4. この導線ではヘッダー有無は自動で `OFF` になるため、必要なら設定で後から切り替える。

### 3.2 ファイルからデータを取り込む

1. 利用者は「開く」を押す。
2. ブラウザのファイル選択ダイアログで `.csv` `.tsv` `.txt` を選ぶ。
3. インポートウィザードが開き、文字コード、区切り文字、推定行数/列数、ファイルサイズが表示される。
4. 利用者は必要に応じて区切り文字、文字コード、ヘッダー扱いを修正する。
5. 「次へ」でプレビューを確認し、「読み込む」で本読込する。
6. 読込後はファイル名バッジと保存先ハンドルが設定される。

### 3.3 読み込んだデータをそのまま編集する

1. セルをクリックして直接編集する。
2. 必要に応じてドラッグ選択、コピー、貼り付け、行列追加、行列削除、並び替えを行う。
3. 編集内容は自動保存対象となる。

### 3.4 現在データを保存・出力する

1. 簡易保存ならツールバーの「保存」「CSVコピー」「CSV保存」を使う。
2. 出力形式や対象を細かく制御したい場合は「エクスポート」を使う。
3. エクスポートではプレビューを見ながら設定を調整し、コピーまたはダウンロードする。

### 3.5 ヘルプ・表示設定を変更する

1. ヘルプボタンでショートカットや注意事項を確認する。
2. 設定ボタンで文字コード、区切り文字、テーマ、言語、アクセントカラーを変更する。
3. 設定変更は即時反映される。

## 4. 画面別詳細仕様

### 4.1 ヘッダー

| 項目 | 内容 |
| --- | --- |
| アプリタイトル | `ArtCSV` 固定表示 |
| ファイル名バッジ | ファイルから読込または保存先確定後に表示。未設定時は非表示 |
| 行数 | ヘッダー行を除いた件数を表示 |
| 列数 | 現在の列数を表示 |
| サイズ | 現在のデータを現行区切り文字と現行文字コードで出力したときの概算サイズを表示 |
| 形式 | 現在の区切り文字に連動して `CSV` `TSV` `SSV` `TXT` 相当の表記に切り替わる |
| ヘルプボタン | 情報＆ヘルプモーダルを開く |
| 設定ボタン | 設定モーダルを開く |

### 4.2 検索バー

| 項目 | 内容 |
| --- | --- |
| 検索入力 | 即時検索対象。検索履歴の候補表示あり |
| 置換入力 | 置換文字列入力 |
| 置換 | 現在の一致位置のみを置換 |
| すべて置換 | 現在の検索条件に一致する全セルを一括置換 |
| 前へ / 次へ | 一致結果を巡回移動 |
| 結果表示 | `検索中...`、`一致なし`、`n / total件` を表示 |
| 詳細 | 詳細検索パネルの開閉 |

詳細検索パネルの設定項目は以下の通り。

| 設定 | 挙動 |
| --- | --- |
| 大文字小文字を区別 | 文字列比較を case-sensitive にする |
| 完全一致 | セル全体一致に限定する |
| 正規表現 | 入力を正規表現として解釈する |
| フィルタ（該当行のみ表示） | 一致した行のみを表示対象にする |
| 対象列 | 全列または特定列に検索範囲を限定する |

検索に関するUI仕様は以下の通り。

- 入力のたびに再検索する。
- `Enter` で検索履歴に保存しつつ次の一致へ移動する。
- `Shift + Enter` で前の一致へ移動する。
- 正規表現が不正な場合は結果欄にエラー表示する。
- データ量が多い場合は検索処理を分割実行し、UIが固まらないようにする。
- 非常に大きいデータでは、全一致セルの常時ハイライトを省略し、現在位置中心の強調に寄せる。

### 4.3 ツールバー

| ボタン | 役割 |
| --- | --- |
| 開く | ファイル選択を起動し、インポートウィザードへ進む |
| 保存 | 読み込んだファイルがある場合は上書き保存。ない場合は保存先選択 |
| Undo | 直前操作を元に戻す |
| Redo | Undoした操作をやり直す |
| 貼り付け | クリップボード内容を新規データとして取り込む |
| 行追加 | 末尾に空行を追加 |
| 列追加 | 末尾に空列を追加 |
| `CSVコピー` など | 現在の区切り文字形式で全データをクリップボードに出力 |
| `CSV保存` など | 現在の区切り文字形式で全データをダウンロード |
| エクスポート | 詳細出力モーダルを開く |
| クリア | 全データ削除確認モーダルを開く |

補足仕様は以下の通り。

- コピー、保存ボタンのラベルは現在の区切り文字に応じて動的に変わる。
- 保存ボタンのラベルは、保存先ファイルハンドルの有無によって「保存」または「上書き保存」の意味合いになる。

### 4.4 空状態ペーストエリア

| 項目 | 内容 |
| --- | --- |
| 主メッセージ | クリック貼り付けの案内 |
| 補助文言 | `Ctrl+V` でも貼り付け可能と案内 |
| 対応形式案内 | CSV/TSV自動判定と案内 |
| 表示条件 | データ未読込時のみ |

仕様上の注意点は以下の通り。

- クリック操作はクリップボード読み取りAPIを使用する。
- クリップボード権限が使えない場合はエラートーストを表示する。
- 自動判定は実質的にカンマ区切りとタブ区切りの判別であり、セミコロンやスペース区切りはこの導線では自動補正されない。

### 4.5 データテーブル

| 項目 | 内容 |
| --- | --- |
| 表示形式 | 1画面内のスクロールテーブル |
| 上部ヘッダー | スクロールしても固定表示 |
| 左側行番号列 | 横スクロールしても固定表示 |
| セル | `contenteditable` による直接編集 |
| 列幅 | 列ごとにリサイズ可能 |
| 行番号 | ヘッダー無効時は1始まり、ヘッダー有効時はデータ部の1始まり |

テーブル表示仕様は以下の通り。

- データが存在しないときはテーブルを完全に隠す。
- テーブルの高さは最大 `70vh` のスクロール領域とする。
- 行数が多い場合は仮想スクロール表示へ切り替え、見えている近傍行のみDOM描画する。
- 空行は取込時に除外される。
- 行ごとの列数が不足している場合は最大列数まで空文字で補完する。

### 4.6 ヘッダー行の扱い

| 項目 | 内容 |
| --- | --- |
| ヘッダーOFF | 1行目も通常データとして編集対象に含む。列見出しは `1, 2, 3...` を表示 |
| ヘッダーON | 1行目を列名として扱い、本文テーブルから除外する |
| 切替導線 | 設定モーダルの「1行目をヘッダーとして扱う」 |

重要なUI仕様は以下の通り。

- ヘッダーON時、列見出しは編集不可。
- 1行目の内容を編集したい場合は、ヘッダーを一度OFFにして1行目セルを編集し、再度ONにする必要がある。
- ヘッダーON時、並び替えや検索対象の本文開始位置は2行目以降になる。

### 4.7 セル編集と移動

| 操作 | 挙動 |
| --- | --- |
| クリック | セルにフォーカス |
| 編集後フォーカス離脱 | 内容を確定 |
| `Tab` | 次セルへ移動 |
| `Shift + Tab` | 前セルへ移動 |
| `Enter` | 下セルへ移動しつつ確定 |
| `Alt + 矢印` | 矢印方向の隣接セルへ移動 |

補足仕様は以下の通り。

- IME変換中はセル移動ショートカットを抑制する。
- フォーカスセルはアクセントカラーで強調表示される。
- 1セルだけクリックした場合は範囲選択としては残さず、通常フォーカスに戻す。

### 4.8 複数セル選択・コピー・貼り付け

| 操作 | 挙動 |
| --- | --- |
| ドラッグ | 矩形範囲選択 |
| `Shift + クリック` | アンカーからの矩形範囲選択 |
| `Ctrl/Cmd + A` | セルフォーカス中のみ本文全体を選択 |
| `Ctrl/Cmd + C` | 選択範囲をTSVとしてコピー |
| `Delete` / `Backspace` | 選択範囲を空文字でクリア |
| `Ctrl/Cmd + V` | 選択範囲またはフォーカスセル起点へ貼り付け |

貼り付け仕様は以下の通り。

- 単一値を複数セル選択へ貼り付けた場合、選択全セルへ同じ値を反映する。
- 複数セルデータを貼り付けた場合、選択範囲の左上またはフォーカスセルを起点に展開する。
- 貼り付け先がテーブル範囲外になる分は切り捨てる。
- データ未読込時のペーストは新規データ取込として扱う。

### 4.9 列操作

列ヘッダー左クリックで列メニューを開く。右クリックメニューは使わない。

| 機能 | 挙動 |
| --- | --- |
| 昇順に並び替え | 対象列で昇順ソート |
| 降順に並び替え | 対象列で降順ソート |
| フィルタ | 列フィルタパネルを開く |
| 左に列を追加 | 対象列の左へ空列挿入 |
| 右に列を追加 | 対象列の右へ空列挿入 |
| 列をコピー | 対象列を改行区切りでコピー |
| 列を複製 | 対象列の右隣へ複製挿入 |
| 列を削除 | 確認後に削除 |
| ドラッグ移動 | ハンドルドラッグで列順序を変更 |
| 列幅変更 | リサイズハンドルのドラッグで変更 |

並び替え仕様は以下の通り。

- ヘッダーON時は1行目を除いて並び替える。
- 数値として解釈できる値同士は数値比較する。
- それ以外はロケール比較で文字列ソートする。
- 大量データ時は非同期化して応答性を維持する。

削除に関する補足は以下の通り。

- 最後の1列を削除すると、データ全体を空状態へ戻す。

### 4.10 行操作

行番号セルまたは行メニューボタンで行メニューを開く。

| 機能 | 挙動 |
| --- | --- |
| 上に行を追加 | 対象行の直前へ空行挿入 |
| 下に行を追加 | 対象行の直後へ空行挿入 |
| 行をコピー | 対象行をTSVとしてコピー |
| 行を複製 | 対象行の直下へ複製挿入 |
| 行を削除 | 確認後に削除 |
| ドラッグ移動 | 行メニューボタンのドラッグで行順序を変更 |

補足仕様は以下の通り。

- ヘッダーON時、1行目は本文に出ないため行メニュー対象にならない。
- 最後の1行を削除すると、データ全体を空状態へ戻す。

### 4.11 列フィルタパネル

| セクション | 内容 |
| --- | --- |
| 値で絞り込み | 値一覧のチェックボックスで対象値を選択 |
| 値検索 | 値一覧の表示候補を絞り込む |
| すべて選択 / すべて解除 | 値一覧のチェック状態を一括変更 |
| 条件で絞り込み | 演算子と値を使った条件式を追加 |
| AND / OR | 同一列内の複数条件の結合方法 |
| フィルタ解除 | 現列フィルタを解除 |
| 絞り込む | 条件を確定して適用 |

条件演算子は以下の通り。

| 演算子 | 意味 |
| --- | --- |
| 含む | 部分一致 |
| 含まない | 部分不一致 |
| 等しい | 完全一致 |
| 等しくない | 完全不一致 |
| 以上 | 数値比較優先、失敗時は文字列比較 |
| 以下 | 数値比較優先、失敗時は文字列比較 |
| 空 | 空文字のみ |
| 非空 | 空文字以外 |

フィルタ結合仕様は以下の通り。

- 同一列では「値一覧条件」と「条件式」の両方を満たす必要がある。
- 複数条件式は AND または OR を選べる。
- 複数列にフィルタがある場合、列間はANDで結合される。
- ヘッダーON時の値一覧集計と判定対象は2行目以降。

### 4.12 インポートウィザード

ウィザードは3ステップ表示だが、実際のファイル選択はブラウザのファイルピッカーで先に行うため、アプリ内UIは判定ステップ相当から開始する。

| ステップ | 内容 |
| --- | --- |
| 選択 | ブラウザのファイル選択ダイアログ |
| 判定 | 文字コード、区切り文字、先頭行数/列数、ファイルサイズの表示と補正 |
| 確認 | 指定条件でのプレビュー確認 |

判定ステップの仕様は以下の通り。

- 文字コードは BOM、文字コード推定、UTF-16のヒューリスティックを使って初期値を決める。
- 区切り文字の自動判定はカンマとタブ中心で行う。
- セミコロン、スペース区切りは利用者が手動変更できる。
- ヘッダー扱いを事前に切り替えられる。

確認ステップの仕様は以下の通り。

- プレビュー行数は `10 / 20 / 50` 行から選択できる。
- ヘッダーON時は1行目を見出しとして表示する。
- ヘッダーOFF時は自動列名を見出し表示する。

### 4.13 保存・簡易出力

| 機能 | 挙動 |
| --- | --- |
| 保存 | 既存ファイルハンドルがあれば上書き、なければ保存先選択 |
| `CSVコピー` など | 現在の区切り文字、改行、クオート設定で全データ出力 |
| `CSV保存` など | 現在の区切り文字、改行、クオート、文字コード、BOM設定で全データをダウンロード |

簡易出力の仕様は以下の通り。

- 出力対象は常に全データである。
- ヘッダーON時はヘッダー行を含めて出力する。
- ヘッダー除外や選択範囲出力は簡易出力ではできず、エクスポートモーダルを使う。
- 文字コード出力に必要な外部変換ライブラリが使えない場合はUTF-8へフォールバックする。

### 4.14 エクスポートモーダル

| 項目 | 選択肢 |
| --- | --- |
| 出力形式 | CSV、TSV、TXT、JSON、NDJSON、HTML Table、Markdown Table |
| 文字コード | UTF-8、Shift-JIS、EUC-JP、ISO-2022-JP、UTF-16 LE、UTF-16 BE |
| BOM | ON/OFF |
| 改行コード | CRLF、LF |
| クオート | 自動、すべて、なし |
| 出力対象 | 全件、フィルタ結果のみ、選択範囲のみ |
| ヘッダー行を含める | ON/OFF |
| 空行を除外する | ON/OFF |
| 空列を除外する | ON/OFF |

形式ごとのUI差分は以下の通り。

| 形式 | 表示される設定 |
| --- | --- |
| CSV / TSV / TXT | 文字コード、BOM、改行、クオートを表示 |
| NDJSON | 改行を表示 |
| JSON / HTML / Markdown | 文字コード、BOM、改行、クオートは非表示 |

出力対象の仕様は以下の通り。

| 対象 | 挙動 |
| --- | --- |
| 全件 | 本文全行を対象 |
| フィルタ結果のみ | 列フィルタと検索フィルタの両方を反映した可視行を対象 |
| 選択範囲のみ | 選択範囲の外接矩形を対象として出力 |

選択範囲出力の重要仕様は以下の通り。

- 疎な選択でも、出力対象は選択セル集合そのものではなく、最小矩形範囲になる。
- ヘッダーを含める場合は、選択列に対応する1行目の値を見出しとして採用する。
- 選択範囲が1行目を含んでいても、ヘッダー行はデータ行と重複しないよう調整される。

プレビュー仕様は以下の通り。

- 最大 `200` 行までプレビューする。
- 条件変更時は自動再計算する。
- フィルタや選択が存在しないのにその対象を選んだ場合、ヒント文で状態を表示し、必要に応じて対象を自動補正する。

### 4.15 設定モーダル

| 項目 | 役割 |
| --- | --- |
| 文字コード | 保存・簡易出力の既定文字コード |
| BOM | 対応文字コード時のみ有効 |
| 区切り文字 | 簡易コピー/簡易保存/表示形式ラベルに反映 |
| 改行コード | 簡易出力時の改行形式 |
| クオート | 簡易出力時の囲み規則 |
| 1行目をヘッダーとして扱う | テーブル表示と検索・集計起点を切り替える |
| テーマ | ライト / ダーク |
| 表示言語 | 日本語 / English |
| アクセントカラー | 6色のスウォッチから選択 |

設定UIの仕様は以下の通り。

- テーマ変更は `document.documentElement.style.colorScheme` を直接切り替える。
- 言語変更は画面全体に即時反映される。
- アクセントカラー変更は主要ボタン、フォーカス、ハイライトの基調色に反映される。
- BOMチェックは対応文字コード以外では無効化される。

## 9. パフォーマンス前提のUI仕様

| 条件 | UI上の扱い |
| --- | --- |
| 可視行数が多い | 仮想スクロールで描画量を削減 |
| 大量検索 | 分割処理またはWeb Workerで非同期検索 |
| 大量フィルタ | Web Workerで値集計・条件評価 |
| 大量統計計算 | サイズ計算を非同期実行し、必要時は `計算中...` 表示 |
| 大量ソート・一括置換 | 非同期実行して操作中トーストを表示 |

## 10. 操作上の制約

| 項目 | 制約 |
| --- | --- |
| ブラウザ右クリック | 無効化されている |
| 列・行メニュー | 左クリック起点の専用UIで操作する |
| ヘッダー編集 | ヘッダーON中は直接編集不可 |
| セミコロン/スペース自動判定 | ファイル取込では手動補正前提、貼り付け取込では非対応寄り |
| 選択範囲エクスポート | 疎な選択そのものではなく矩形範囲を出力する |
